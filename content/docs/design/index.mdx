## 設計方針

デザインシステムのトークンは CSS 変数として定義されており、Tailwind CSS の utility クラスにマップされています。

```css
/* :root に定義 */
--primary: oklch(0.205 0 0);
--radius: 0.625rem;
```

```tsx
/* Tailwind で利用 */
<div className="bg-primary text-primary-foreground rounded-lg" />
```

コンポーネントはすべてこれらのトークンを参照するため、`:root` の値を変更するだけでテーマ全体を切り替えることができます。

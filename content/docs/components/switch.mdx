### Description

<ComponentPreview styleName="base" name="switch-description" />

### Choice Card

`FieldLabel` が `Field` 全体を包括するカードスタイルの選択。クリック可能なカードパターン。

<ComponentPreview styleName="base" name="switch-choice-card" />

### Disabled

`Switch` に `disabled` を付けて無効化します。スタイリングには `Field` に `data-disabled` を付けます。

<ComponentPreview styleName="base" name="switch-disabled" />

### Invalid

`Switch` に `aria-invalid` を付けて未入力状態を示します。スタイリングには `Field` に `data-invalid` を付けます。

<ComponentPreview styleName="base" name="switch-invalid" />

### Size

`size` でスイッチのサイズを変更できます。

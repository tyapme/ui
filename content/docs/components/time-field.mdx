## 使い方

```tsx showLineNumbers
import { TimeFieldInput } from "@/components/ui/time-field"
```

```tsx showLineNumbers
<TimeFieldInput
  value={date}
  onValueChange={setDate}
  aria-label="Time"
/>
```

## Props

### TimeFieldInput

| Prop           | Type                             | Default    | 説明                           |
| -------------- | -------------------------------- | ---------- | ------------------------------ |
| `value`        | `Date \| undefined`              | —          | 現在の時刻値                   |
| `onValueChange`| `(date: Date \| undefined) => void` | —       | 時刻変更時のコールバック       |
| `disabled`     | `boolean`                        | `false`    | 無効化                         |
| `granularity`  | `"hour" \| "minute" \| "second"` | `"minute"` | 表示する精度                   |
| `id`           | `string`                         | —          | フィールドの ID                |
| `aria-label`   | `string`                         | —          | アクセシビリティラベル         |

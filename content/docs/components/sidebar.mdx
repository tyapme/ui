## 構造

- **SidebarProvider** — 折りたたみ状態を管理し、子コンポーネントにコンテキストを提供します。
- **Sidebar** — 折りたたみ対応のメインサイドバーパネル。
- **SidebarHeader** — 上部に固定され、ブランディング・タイトル・ワークスペース切り替え等に使用します。
- **SidebarFooter** — 下部に固定され、ユーザーメニュー・設定・アクション等に使用します。
- **SidebarContent** — ヘッダーとフッターの間のスクロール可能な領域。
- **SidebarGroup** — オプションのラベル・アクション・コンテンツエリア付きで関連ナビゲーションをグループ化します。
- **SidebarMenu** / **SidebarMenuItem** — リンク・バッジ・アクション・ネストサブメニュー向けのメニュー構造。
- **SidebarRail** — 対応時にサイドバー幅を調整するリサイズハンドル。
- **SidebarInset** — `inset` バリアント利用時にメインコンテンツを包括します。
- **SidebarTrigger** — サイドバーの開閉をトグルするコントロール。

## SidebarProvider

`SidebarProvider` コンポーネントは `Sidebar` コンポーネントにコンテキストを提供するために使用します。常にアプリ全体を `SidebarProvider` で包括してください。

### プロパティ

| 名前 | 型 | 説明 |
| -------------- | ------------------------- | -------------------------------------------- |
| `defaultOpen`  | `boolean`                 | サイドバーの初期開く状態。 |
| `open`         | `boolean`                 | サイドバーの開く状態（制御あり）。 |
| `onOpenChange` | `(open: boolean) => void` | サイドバーの開く状態を設定する（制御あり）。 |

### 幅

アプリにサイドバーが 1 つだけの場合、`sidebar.tsx` 内の `SIDEBAR_WIDTH` と `SIDEBAR_WIDTH_MOBILE` 変数で幅を設定できます。

```tsx showLineNumbers title="components/ui/sidebar.tsx"
const SIDEBAR_WIDTH = "16rem"
const SIDEBAR_WIDTH_MOBILE = "18rem"
```

アプリに複数のサイドバーがある場合は、`style` 属性で CSS 変数 `--sidebar-width` と `--sidebar-width-mobile` を使用できます。

```tsx showLineNumbers
<SidebarProvider
  style={
    {
      "--sidebar-width": "20rem",
      "--sidebar-width-mobile": "20rem",
    } as React.CSSProperties
  }
>
  <Sidebar />
</SidebarProvider>
```

### キーボードショートカット

サイドバーのトリガーに Mac では `cmd+b`、Windows では `ctrl+b` を使用します。

```tsx showLineNumbers title="components/ui/sidebar.tsx"
const SIDEBAR_KEYBOARD_SHORTCUT = "b"
```

## Sidebar

折りたたみ対応のサイドバーをレンダリングするメイン `Sidebar` コンポーネント。

### プロパティ

| プロパティ | 型 | 説明 |
| ------------- | --------------------------------- | --------------------------------- |
| `side`        | `left` または `right`             | サイドバーの表示側。 |
| `variant`     | `sidebar`、`floating`、`inset`   | サイドバーのバリアント。 |
| `collapsible` | `offcanvas`、`icon`、`none`      | 折りたたみの状態。 |

| 値 | 説明 |
| ----------- | ------------------------------------------------------------ |
| `offcanvas` | 左右からスライドインする折りたたみ対応サイドバー。 |
| `icon`      | アイコンに縮小するサイドバー。 |
| `none`      | 折りたたみなしのサイドバー。 |

<Callout>
  **注意:** `inset` バリアントを使用する場合は、必ずメインコンテンツを `SidebarInset` コンポーネントで包括してください。
</Callout>

```tsx showLineNumbers
<SidebarProvider>
  <Sidebar variant="inset" />
  <SidebarInset>
    <main>{children}</main>
  </SidebarInset>
</SidebarProvider>
```

## useSidebar

`useSidebar` フックはサイドバーを制御するために使用します。

```tsx showLineNumbers
import { useSidebar } from "@/components/ui/sidebar"

export function AppSidebar() {
  const {
    state,
    open,
    setOpen,
    openMobile,
    setOpenMobile,
    isMobile,
    toggleSidebar,
  } = useSidebar()
}
```

| プロパティ | 型 | 説明 |
| --------------- | ------------------------- | --------------------------------------------- |
| `state`         | `expanded` または `collapsed` | サイドバーの現在の状態。 |
| `open`          | `boolean`                 | サイドバーが開いているかどうか。 |
| `setOpen`       | `(open: boolean) => void` | サイドバーの開いた状態を設定する。 |
| `openMobile`    | `boolean`                 | モバイルでサイドバーが開いているかどうか。 |
| `setOpenMobile` | `(open: boolean) => void` | モバイルでサイドバーの開いた状態を設定する。 |
| `isMobile`      | `boolean`                 | モバイル表示かどうか。 |
| `toggleSidebar` | `() => void`              | サイドバーをトグルする。デスクトップ・モバイル共通。 |

## SidebarGroup

`SidebarGroup` でサイドバー内にセクションを作成します。

`SidebarGroup` には `SidebarGroupLabel`、`SidebarGroupContent`、および任意の `SidebarGroupAction` があります。

```tsx showLineNumbers
<SidebarGroup>
  <SidebarGroupLabel>Application</SidebarGroupLabel>
  <SidebarGroupAction>
    <Plus /> <span className="sr-only">Add Project</span>
  </SidebarGroupAction>
  <SidebarGroupContent></SidebarGroupContent>
</SidebarGroup>
```

`SidebarGroup` を折りたたみ対応にするには、`Collapsible` で包括します。

```tsx showLineNumbers
<Collapsible defaultOpen className="group/collapsible">
  <SidebarGroup>
    <SidebarGroupLabel asChild>
      <CollapsibleTrigger>
        Help
        <ChevronDown className="ml-auto transition-transform group-data-[state=open]/collapsible:rotate-180" />
      </CollapsibleTrigger>
    </SidebarGroupLabel>
    <CollapsibleContent>
      <SidebarGroupContent />
    </CollapsibleContent>
  </SidebarGroup>
</Collapsible>
```

## SidebarMenuButton

`SidebarMenuButton` は `SidebarMenuItem` 内にメニューボタンをレンダリングするために使用します。

}

## 変更履歴

### RTL サポート

前のバージョンからアップグレードする場合、RTL サポートを追加するために以下の変更を適用してください:

<Steps>

<Step>Sidebar コンポーネントに `dir` prop を追加します。</Step>

モバイル向け `SheetContent` に `dir` を渡すため、展開した props に追加します:

```diff
  function Sidebar({
    side = "left",
    variant = "sidebar",
    collapsible = "offcanvas",
    className,
    children,
+   dir,
    ...props
  }: React.ComponentProps<"div"> & {
    side?: "left" | "right"
    variant?: "sidebar" | "floating" | "inset"
    collapsible?: "offcanvas" | "icon" | "none"
  }) {
```

モバイルビューの `SheetContent` に渡します:

```diff
  <Sheet open={openMobile} onOpenChange={setOpenMobile} {...props}>
    <SheetContent
+     dir={dir}
      data-sidebar="sidebar"
      data-slot="sidebar"
      data-mobile="true"
```

<Step>サイドバーコンテナーに `data-side` 属性を追加します。</Step>

Add `data-side={side}` to the sidebar container element:

```diff
  <div
    data-slot="sidebar-container"
+   data-side={side}
    className={cn(
```

<Step>サイドバーコンテナーの位置指定クラスを更新します。</Step>

JavaScript の三項演算ショートカットクラスを CSS の data 属性セレクターに置き換えます:

```diff
  className={cn(
-   "fixed inset-y-0 z-10 hidden h-svh w-(--sidebar-width) transition-[left,right,width] duration-200 ease-linear md:flex",
-   side === "left"
-     ? "left-0 group-data-[collapsible=offcanvas]:left-[calc(var(--sidebar-width)*-1)]"
-     : "right-0 group-data-[collapsible=offcanvas]:right-[calc(var(--sidebar-width)*-1)]",
+   "fixed inset-y-0 z-10 hidden h-svh w-(--sidebar-width) transition-[left,right,width] duration-200 ease-linear md:flex data-[side=left]:left-0 data-[side=right]:right-0 data-[side=left]:group-data-[collapsible=offcanvas]:left-[calc(var(--sidebar-width)*-1)] data-[side=right]:group-data-[collapsible=offcanvas]:right-[calc(var(--sidebar-width)*-1)]",
```

<Step>SidebarRail の位置指定クラスを更新します。</Step>

`SidebarRail` コンポーネントの物理的位置指定を更新します:

```diff
  className={cn(
-   "hover:after:bg-sidebar-border absolute inset-y-0 z-20 hidden w-4 -translate-x-1/2 transition-all ease-linear group-data-[side=left]:-end-4 group-data-[side=right]:start-0 after:absolute after:inset-y-0 after:start-1/2 after:w-[2px] sm:flex",
+   "hover:after:bg-sidebar-border absolute inset-y-0 z-20 hidden w-4 ltr:-translate-x-1/2 rtl:-translate-x-1/2 transition-all ease-linear group-data-[side=left]:-right-4 group-data-[side=right]:left-0 after:absolute after:inset-y-0 after:start-1/2 after:w-[2px] sm:flex",
```

<Step>SidebarTrigger のアイコンに RTL フリップを追加します。</Step>

`SidebarTrigger` のアイコンに `className="rtl:rotate-180"` を付けて RTL モードで反転させます:

```diff
  <Button ...>
-   <PanelLeftIcon />
+   <PanelLeftIcon className="rtl:rotate-180" />
    <span className="sr-only">Toggle Sidebar</span>
  </Button>
```

</Steps>

これらの変更を適用すると、`dir` で方向を設定できます:

```tsx
<Sidebar dir="rtl" side="right">
  {/* ... */}
</Sidebar>
```

サイドバーは LTR ・ RTL の両方のレイアウトで正しく位置決めと操作を実行します。

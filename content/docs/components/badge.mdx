```

```

## 使用例

### Variants

`variant` プロパティでバッジのバリアントを変更します。

<ComponentPreview styleName="base" name="badge-variants" />

### With Icon

バッジの中にアイコンをレンダリングできます。`data-icon="inline-start"` で左側、`data-icon="inline-end"` で右側に配置します。

<ComponentPreview styleName="base" name="badge-icon" />

### With Spinner

バッジの中にスピナーをレンダリングできます。スピナーに `data-icon="inline-start"` または `data-icon="inline-end"` 属性を付けて正しい間隔を確保してください。

<ComponentPreview styleName="base" name="badge-spinner" />

### Link

`render` プロパティを使ってバッジをリンクとしてレンダリングします。

<ComponentPreview styleName="base" name="badge-link" />

### Custom Colors

`Badge` コンポーネントに `bg-green-50 dark:bg-green-800` などのカスタムクラスを追加してカラーをカスタマイズできます。

<ComponentPreview styleName="base" name="badge-colors" />

## API リファレンス

### Badge

`Badge` コンポーネントはバッジまたはバッジのようなコンポーネントを表示します。

| Prop        | Type                                                                          | Default     |
| ----------- | ----------------------------------------------------------------------------- | ----------- |
| `variant`   | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"` | `"default"` |
| `className` | `string`                                                                      | -           |

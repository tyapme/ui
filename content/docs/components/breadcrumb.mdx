## 使い方

```tsx showLineNumbers
import {
  Breadcrumb,
  BreadcrumbItem,
  BreadcrumbLink,
  BreadcrumbList,
  BreadcrumbPage,
  BreadcrumbSeparator,
} from "@/components/ui/breadcrumb"
```

```tsx showLineNumbers
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink render={<a href="/" />}>Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink render={<a href="/components" />}>
        Components
      </BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Breadcrumb</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

## 構成

`Breadcrumb` の構成:

```text
Breadcrumb
└── BreadcrumbList
    ├── BreadcrumbItem
    │   └── BreadcrumbLink
    ├── BreadcrumbSeparator
    ├── BreadcrumbItem
    │   └── BreadcrumbLink
    ├── BreadcrumbSeparator
    └── BreadcrumbItem
        └── BreadcrumbPage
```

## API リファレンス

### Breadcrumb

`Breadcrumb` コンポーネントはすべてのパンくずリスト要素を包むルートナビゲーション要素です。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### BreadcrumbList

`BreadcrumbList` コンポーネントはパンくずリストアイテムの順序付きリストを表示します。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### BreadcrumbItem

`BreadcrumbItem` コンポーネントは個別のパンくずリストアイテムを包みます。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### BreadcrumbLink

`BreadcrumbLink` コンポーネントはパンくずリスト内のクリック可能なリンクを表示します。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### BreadcrumbPage

`BreadcrumbPage` コンポーネントはパンくずリストの現在のページ（クリック不可）を表示します。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### BreadcrumbSeparator

`BreadcrumbSeparator` コンポーネントはパンくずリストアイテム間のセパレーターを表示します。カスタムな children を渡すとデフォルトのセパレーターアイコンを上書きできます。

| Prop        | Type              | Default |
| ----------- | ----------------- | ------- |
| `children`  | `React.ReactNode` | -       |
| `className` | `string`          | -       |

### BreadcrumbEllipsis

`BreadcrumbEllipsis` コンポーネントは折りたたまれたパンくずリストアイテムのエリプシス表示を可能にします。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

## チェック状態

革制されないチェックボックスには `defaultChecked` を、状態を制御する場合は `checked` と `onCheckedChange` を使用します。

```tsx showLineNumbers
import * as React from "react"

export function Example() {
  const [checked, setChecked] = React.useState(false)

  return <Checkbox checked={checked} onCheckedChange={setChecked} />
}
```

## 無効状態

チェックボックスに `aria-invalid`、フィールドラッパーに `data-invalid` を設定して無効スタイルを表示します。

## 使い方

```tsx showLineNumbers
import {
  InputGroup,
  InputGroupAddon,
  InputGroupButton,
  InputGroupInput,
  InputGroupText,
  InputGroupTextarea,
} from "@/components/ui/input-group"
```

```tsx showLineNumbers
<InputGroup>
  <InputGroupInput placeholder="Search..." />
  <InputGroupAddon>
    <SearchIcon />
  </InputGroupAddon>
</InputGroup>
```

## 構成

`InputGroup` の構成:

```text
InputGroup
├── InputGroupInput or InputGroupTextarea
├── InputGroupAddon
├── InputGroupButton
└── InputGroupText
```

## 配置

`InputGroupAddon` の `align` でアドオンの入力に対する位置を制御します。

<Callout>
  フォーカス管理を正しく動作させるため、`InputGroupAddon` は DOM 上で常に `InputGroupInput` または `InputGroupTextarea` の後に配置してください。視覚的な位置小けに `align` を使用してください。
</Callout>

## API リファレンス

### InputGroup

入力とアドオンを包括するメインコンポーネント。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` |         |

```tsx
<InputGroup>
  <InputGroupInput />
  <InputGroupAddon />
</InputGroup>
```

### InputGroupAddon

入力の横にアイコン、テキスト、ボタン、その他のコンテンツを表示します。

<Callout icon={<IconInfoCircle />} title="フォーカスナビゲーション">
  正しいフォーカスナビゲーションのため、`InputGroupAddon` は常に入力の後に配置してください。`align` で表示位置を調整します。
</Callout>

| Prop        | Type                                                             | Default          |
| ----------- | ---------------------------------------------------------------- | ---------------- |
| `align`     | `"inline-start" \| "inline-end" \| "block-start" \| "block-end"` | `"inline-start"` |
| `className` | `string`                                                         |                  |

```tsx
<InputGroupAddon align="inline-end">
  <SearchIcon />
</InputGroupAddon>
```

**`<InputGroupInput />` には `inline-start` または `inline-end`、`<InputGroupTextarea />` には `block-start` または `block-end` を使用します。**

`InputGroupAddon` には複数の `InputGroupButton` やアイコンを含めることができます。

```tsx
<InputGroupAddon>
  <InputGroupButton>Button</InputGroupButton>
  <InputGroupButton>Button</InputGroupButton>
</InputGroupAddon>
```

### InputGroupButton

入力グループ内にボタンを表示します。

| Prop        | Type                                                                          | Default   |
| ----------- | ----------------------------------------------------------------------------- | --------- |
| `size`      | `"xs" \| "icon-xs" \| "sm" \| "icon-sm"`                                      | `"xs"`    |
| `variant`   | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"ghost"` |
| `className` | `string`                                                                      |           |

```tsx
<InputGroupButton>Button</InputGroupButton>
<InputGroupButton size="icon-xs" aria-label="Copy">
  <CopyIcon />
</InputGroupButton>
```

### InputGroupInput

入力グループ構築時に `<Input />` の代替となるコンポーネント。入力グループ向けスタイルが先に適用され、`data-slot="input-group-control"` でフォーカス状態を管理します。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` |         |

その他の props は内部の `<Input />` コンポーネントにそのまま渡されます。

```tsx
<InputGroup>
  <InputGroupInput placeholder="テキストを入力..." />
  <InputGroupAddon>
    <SearchIcon />
  </InputGroupAddon>
</InputGroup>
```

### InputGroupTextarea

入力グループ構築時に `<Textarea />` の代替となるコンポーネント。テキストエリアグループ向けスタイルが先に適用され、`data-slot="input-group-control"` でフォーカス状態を管理します。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` |         |

その他の props は内部の `<Textarea />` コンポーネントにそのまま渡されます。

```tsx
<InputGroup>
  <InputGroupTextarea placeholder="メッセージを入力..." />
  <InputGroupAddon align="block-end">
    <InputGroupButton>送信</InputGroupButton>
  </InputGroupAddon>
</InputGroup>
```

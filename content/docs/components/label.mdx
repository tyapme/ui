## 使い方

```tsx
import { Label } from "@/components/ui/label"
```

```tsx
<Label htmlFor="email">Your email address</Label>
```

## Field 内の Label

フォームフィールドには、`FieldLabel`、`FieldDescription`、`FieldError` を内包する [Field](/docs/components/base/field) コンポーネントを使用してください。

```tsx
<Field>
  <FieldLabel htmlFor="email">Your email address</FieldLabel>
  <Input id="email" />
</Field>
```

**Charts** のご紹介。アプリにコピー&ペーストで使えるチャートコンポーネントのコレクション。

チャートはデフォルトで美しく見えます。他のコンポーネントとも整合され、プロジェクトに合わせて完全にカスタマイズできます。

## コンポーネント

内部に [Recharts](https://recharts.org/) を使用しています。

`chart` コンポーネントはコンポジッションを意識して設計されています。**Recharts コンポーネントでチャートを構築し、`ChartTooltip` などのカスタムコンポーネントは必要なときに必要な場所でだけ持ち込ます**。

```tsx showLineNumbers /ChartContainer/ /ChartTooltipContent/
import { Bar, BarChart } from "recharts"

import { ChartContainer, ChartTooltipContent } from "@/components/ui/chart"

export function MyChart() {
  return (
    <ChartContainer>
      <BarChart data={data}>
        <Bar dataKey="value" />
        <ChartTooltip content={<ChartTooltipContent />} />
      </BarChart>
    </ChartContainer>
  )
}
```

Recharts をラップしていません。つまり、抽象化に縛り付けられません。Recharts の新バージョンがリリースされた際は、公式のアップグレード手順に従ってチャートを更新できます。

**コンポーネントはあなたのものです**。

## Recharts v3 へのアップデート

古いチャートコードを Recharts v3 に更新する場合:

- CSS 変数のチャートトークンを参照する際は `hsl(var(--chart-1))` の代わりに `var(--chart-1)` を使用してください。
- 初期ツールチップ状態にのみ `ChartTooltip.defaultIndex` を使用してください。永続的なアクティブ形状はチャート自身の state で管理してください。
- 親の `<BarChart>` に `layout` が既に定義されている場合は `<Bar>` から `layout` を削除してください。
- `ResponsiveContainer` が初回レンダリングでサイズを計測できるよう、`ChartContainer` に `height`、`min-h-*`、または `aspect-*` を必ず設定してください。

## チャート設定

チャート設定は、チャートのラベル、アイコン、カラーを定義する場所です。

チャートデータから意図的に分離しています。

チャート間で設定とカラートークンを共有できます。データやカラートークンがリモートや別形式にある場合にも対応できます。

```tsx showLineNumbers /ChartConfig/
import { Monitor } from "lucide-react"

import { type ChartConfig } from "@/components/ui/chart"

const chartConfig = {
  desktop: {
    label: "Desktop",
    icon: Monitor,
    // A color like 'hsl(220, 98%, 61%)' or 'var(--color-name)'
    color: "#2563eb",
    // OR a theme object with 'light' and 'dark' keys
    theme: {
      light: "#2563eb",
      dark: "#dc2626",
    },
  },
} satisfies ChartConfig
```

### カラーの使用

チャートでテーマカラーを使用するには、`var(--color-KEY)` 形式で参照します。

#### コンポーネント

```tsx

```

#### チャートデータ


## ツールチップ

チャートのツールチップにはラベル、名前、インジケーター、値が含まれます。これらを組み合わせてツールチップをカスタマイズできます。

<ComponentPreview styleName="base" name="chart-tooltip" hideCode />

`hideLabel`、`hideIndicator` で各要素の表示/非表示を切り替え、`indicator` でインジケーターのスタイルをカスタマイズできます。

`labelKey` と `nameKey` でツールチップのラベルと名前にカスタムキーを指定できます。

`<ChartTooltip>` と `<ChartTooltipContent>` コンポーネントを使ってチャートにカスタムツールチップを追加できます。

```tsx title="components/example-chart.tsx"
import { ChartTooltip, ChartTooltipContent } from "@/components/ui/chart"
```

```tsx title="components/example-chart.tsx"
<ChartTooltip content={<ChartTooltipContent />} />
```

### プロパティ

ツールチップのカスタマイズに使用する props。

| プロパティ | 型 | 説明 |
| :-------------- | :----------------------- | :------------------------------------------- |
| `labelKey`      | string                   | ラベルに使用する設定またはデータキー。 |
| `nameKey`       | string                   | 名前に使用する設定またはデータキー。 |
| `indicator`     | `dot` `line` or `dashed` | ツールチップのインジケータースタイル。 |
| `hideLabel`     | boolean                  | ラベルを非表示にするかどうか。 |
| `hideIndicator` | boolean                  | インジケーターを非表示にするかどうか。 |

### カラー

カラーはチャート設定から自動的に参照されます。

### カスタム

ツールチップのラベルと名前にカスタムキーを使用するには、`labelKey` と `nameKey` を使用します。

```tsx showLineNumbers /browser/
const chartData = [
  { browser: "chrome", visitors: 187, fill: "var(--color-chrome)" },
  { browser: "safari", visitors: 200, fill: "var(--color-safari)" },
]

const chartConfig = {
  visitors: {
    label: "Total Visitors",
  },
  chrome: {
    label: "Chrome",
    color: "var(--chart-1)",
  },
  safari: {
    label: "Safari",
    color: "var(--chart-2)",
  },
} satisfies ChartConfig
```

```tsx title="components/example-chart.tsx"
<ChartTooltip
  content={<ChartTooltipContent labelKey="visitors" nameKey="browser" />}
/>
```

ラベルに `Total Visitors`、ツールチップの名前に `Chrome` と `Safari` が使用されます。

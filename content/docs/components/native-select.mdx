## 構成

### シンプル

`NativeSelect` の直下にオプションを配置する（`NativeSelectOptGroup` なし）。

```text
NativeSelect
├── NativeSelectOption
├── NativeSelectOption
├── NativeSelectOption
└── NativeSelectOption
```

### グループあり

`NativeSelectOptGroup` でオプションをカテゴリ分けします。

```text
NativeSelect
├── NativeSelectOptGroup
│   ├── NativeSelectOption
│   └── NativeSelectOption
└── NativeSelectOptGroup
    ├── NativeSelectOption
    └── NativeSelectOption
```

## NativeSelect vs Select

- ネイティブブラウザの動作、パフォーマンス重視、またはモバイル最適化ドロップダウンを必要とする場合は `NativeSelect` を使用します。
- カスタムスタイリング、アニメーション、または複雑なインタラクションを必要とする場合は `Select` を使用します。

## API リファレンス

### NativeSelect

ネイティブ HTML の select 要素を包括するメインセレクトコンポーネント。

```tsx
<NativeSelect>
  <NativeSelectOption value="option1">Option 1</NativeSelectOption>
  <NativeSelectOption value="option2">Option 2</NativeSelectOption>
</NativeSelect>
```

### NativeSelectOption

セレクト内の個別のオプションを表します。

| Prop       | Type      | Default |
| ---------- | --------- | ------- |
| `value`    | `string`  |         |
| `disabled` | `boolean` | `false` |

### NativeSelectOptGroup

関連するオプションをグループ化して整理します。

| Prop       | Type      | Default |
| ---------- | --------- | ------- |
| `label`    | `string`  |         |
| `disabled` | `boolean` | `false` |

```tsx
<NativeSelectOptGroup label="Fruits">
  <NativeSelectOption value="apple">Apple</NativeSelectOption>
  <NativeSelectOption value="banana">Banana</NativeSelectOption>
</NativeSelectOptGroup>
```

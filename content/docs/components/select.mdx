## 使い方

```tsx showLineNumbers
import {
  Select,
  SelectContent,
  SelectGroup,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@/components/ui/select"
```

```tsx showLineNumbers
const items = [
  { label: "Light", value: "light" },
  { label: "Dark", value: "dark" },
  { label: "System", value: "system" },
]

<Select items={items}>
  <SelectTrigger className="w-[180px]">
    <SelectValue placeholder="Theme" />
  </SelectTrigger>
  <SelectContent>
    <SelectGroup>
      {items.map((item) => (
        <SelectItem key={item.value} value={item.value}>
          {item.label}
        </SelectItem>
      ))}
    </SelectGroup>
  </SelectContent>
</Select>
```

## 構成

`Select` の構成:

```text
Select
├── SelectTrigger
│   └── SelectValue
└── SelectContent
    ├── SelectGroup
    │   ├── SelectLabel
    │   ├── SelectItem
    │   └── SelectItem
    ├── SelectSeparator
    └── SelectGroup
        ├── SelectLabel
        ├── SelectItem
        └── SelectItem
```

## 使用例

### 選択項目をトリガーに履わせる

`SelectContent` の `alignItemWithTrigger` で選択項目をトリガーに履わせるかどうかを制御します。`true`（デフォルト）の場合、選択中の項目がトリガーの上に表示されます。`false` の場合、ポップアップがトリガーの端に履わされます。

### グループ

`SelectGroup`、`SelectLabel`、`SelectSeparator` で項目を整理します。

### エラー

`Field` に `data-invalid`、`SelectTrigger` に `aria-invalid` を付けてエラー状態を表示します。

```tsx showLineNumbers /data-invalid/ /aria-invalid/
<Field data-invalid>
  <FieldLabel>Fruit</FieldLabel>
  <SelectTrigger aria-invalid>
    <SelectValue />
  </SelectTrigger>
</Field>
```

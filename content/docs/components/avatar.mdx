## 使い方


```tsx showLineNumbers
<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" />
  <AvatarFallback>CN</AvatarFallback>
</Avatar>
```

## 構成

`Avatar` の構成:

```text
Avatar
├── AvatarImage
├── AvatarFallback
└── AvatarBadge
```

`AvatarGroup` の構成:

```text
AvatarGroup
├── Avatar
│   ├── AvatarImage
│   ├── AvatarFallback
│   └── AvatarBadge
├── Avatar
│   ├── AvatarImage
│   ├── AvatarFallback
│   └── AvatarBadge
└── AvatarGroupCount
```

`className` プロパティでカスタムカラーやサイズなどのスタイルをバッジに追加できます。

```tsx showLineNumbers
<Avatar>
  <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
  <AvatarFallback>CN</AvatarFallback>
  <AvatarBadge className="bg-green-600 dark:bg-green-800" />
</Avatar>
```

## API リファレンス

### Avatar

`Avatar` コンポーネントはアバター画像とフォールバックを包むルートコンポーネントです。

| Prop        | Type                        | Default     |
| ----------- | --------------------------- | ----------- |
| `size`      | `"default" \| "sm" \| "lg"` | `"default"` |
| `className` | `string`                    | -           |

### AvatarImage

`AvatarImage` コンポーネントはアバター画像を表示します。Base UI Avatar Image のすべての props を受け付けます。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `src`       | `string` | -       |
| `alt`       | `string` | -       |
| `className` | `string` | -       |

### AvatarFallback

`AvatarFallback` コンポーネントは画像の読み込みに失敗した際にフォールバックを表示します。Base UI Avatar Fallback のすべての props を受け付けます。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### AvatarBadge

`AvatarBadge` コンポーネントはアバターにバッジインジケーターを表示します。逆に右下に配置されます。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### AvatarGroup

`AvatarGroup` コンポーネントは重なりスタイルでアバターのグループを表示します。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

### AvatarGroupCount

`AvatarGroupCount` コンポーネントはアバターグループにカウント表示を追加します。残りのアバター数を表示するのに使います。

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` | -       |

Base UI Avatar props の詳細は [Base UI ドキュメント](https://base-ui.com/react/components/avatar#api-reference) を参照してください。

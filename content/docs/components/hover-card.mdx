## 使い方

```tsx showLineNumbers
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@/components/ui/hover-card"
```

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger>Hover</HoverCardTrigger>
  <HoverCardContent>
    TYAP.ME UI component notes for product surfaces.
  </HoverCardContent>
</HoverCard>
```

## 構成

`HoverCard` の構成:

```text
HoverCard
├── HoverCardTrigger
└── HoverCardContent
```

## トリガーの遅延

トリガーの `delay` と `closeDelay` でカードの開閉タイミングを制御します。

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger delay={100} closeDelay={200}>
    Hover
  </HoverCardTrigger>
  <HoverCardContent>Content</HoverCardContent>
</HoverCard>
```

## 配置

`HoverCardContent` の `side` と `align` で表示位置を制御します。

```tsx showLineNumbers
<HoverCard>
  <HoverCardTrigger>Hover</HoverCardTrigger>
  <HoverCardContent side="top" align="start">
    Content
  </HoverCardContent>
</HoverCard>
```

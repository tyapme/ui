<TabsContent value="cli">

<Steps className="mb-0 pt-2">

<Step>以下のコマンドを実行します:</Step>

```bash
npx shadcn@latest add https://ui.tyap.me/r/styles/base/sonner.json
```

<Step>Toaster コンポーネントを追加します。</Step>

```tsx title="app/layout.tsx" {1,9} showLineNumbers
import { Toaster } from "@/components/ui/sonner"

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <head />
      <body>
        <main>{children}</main>
        <Toaster />
      </body>
    </html>
  )
}
```

</Steps>

</TabsContent>

<Steps className="mb-0 pt-2">

npm install sonner next-themes

<Step>以下のコードをプロジェクトにコピーします。</Step>

import { Toaster } from "@/components/ui/sonner"

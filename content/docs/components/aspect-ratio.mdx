## 使い方

```tsx showLineNumbers
import { AspectRatio } from "@/components/ui/aspect-ratio"
```

```tsx showLineNumbers
<AspectRatio ratio={16 / 9}>
  <Image src="..." alt="Image" className="rounded-md object-cover" />
</AspectRatio>
```

## 使用例

### Square

`ratio={1 / 1}` を使った正方形のアスペクト比コンポーネント。画像を正方形で表示するのに便利です。

## API リファレンス

### AspectRatio

`AspectRatio` コンポーネントは指定したアスペクト比でコンテンツを表示します。

| Prop        | Type     | Default | Required |
| ----------- | -------- | ------- | -------- |
| `ratio`     | `number` | -       | Yes      |
| `className` | `string` | -       | No       |

詳細は [Base UI ドキュメント](https://base-ui.com/react/components/aspect-ratio#api-reference) を参照してください。

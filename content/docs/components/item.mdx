## 使い方

```tsx showLineNumbers
import {
  Item,
  ItemActions,
  ItemContent,
  ItemDescription,
  ItemMedia,
  ItemTitle,
} from "@/components/ui/item"
```

```tsx showLineNumbers
<Item>
  <ItemMedia variant="icon">
    <Icon />
  </ItemMedia>
  <ItemContent>
    <ItemTitle>Title</ItemTitle>
    <ItemDescription>Description</ItemDescription>
  </ItemContent>
  <ItemActions>
    <Button>Action</Button>
  </ItemActions>
</Item>
```

## 構成

`Item` の構成:

```text
ItemGroup
└── Item
    ├── ItemHeader
    ├── ItemMedia
    ├── ItemContent
    │   ├── ItemTitle
    │   └── ItemDescription
    ├── ItemActions
    └── ItemFooter
```

## Item vs Field

チェックボックス、入力欄、ラジオ、セレクトなどのフォーム入力を表示する場合は `Field` を使用します。

タイトル、説明、アクションなどのコンテンツだけを表示する場合は `Item` を使用します。

## API リファレンス

### Item

メディア、タイトル、説明、アクションを持つコンテンツを表示するメインコンポーネント。

| Prop      | Type                                | Default     |
| --------- | ----------------------------------- | ----------- |
| `variant` | `"default" \| "outline" \| "muted"` | `"default"` |
| `size`    | `"default" \| "sm" \| "xs"`         | `"default"` |
| `render`  | `React.ReactElement`                |             |

### ItemGroup

一賫したスタイリングで関連アイテムをまとめるコンテナー。

```tsx
<ItemGroup>
  <Item />
  <Item />
</ItemGroup>
```

### ItemSeparator

グループ内のアイテム間の区切り線。

```tsx
<ItemGroup>
  <Item />
  <ItemSeparator />
  <Item />
</ItemGroup>
```

### ItemMedia

アイコン、画像、アバターなどのメディアコンテンツを表示します。

| Prop      | Type                             | Default     |
| --------- | -------------------------------- | ----------- |
| `variant` | `"default" \| "icon" \| "image"` | `"default"` |

```tsx
<ItemMedia variant="icon">
  <Icon />
</ItemMedia>
```

```tsx
<ItemMedia variant="image">
  <img src="..." alt="..." />
</ItemMedia>
```

### ItemContent

アイテムのタイトルと説明を包括します。

```tsx
<ItemContent>
  <ItemTitle>Title</ItemTitle>
  <ItemDescription>Description</ItemDescription>
</ItemContent>
```

### ItemTitle

アイテムのタイトルを表示します。

```tsx
<ItemTitle>Item Title</ItemTitle>
```

### ItemDescription

アイテムの説明を表示します。

```tsx
<ItemDescription>Item description</ItemDescription>
```

### ItemActions

アクションボタンやその他のインタラクティブ要素のコンテナー。

```tsx
<ItemActions>
  <Button>Action</Button>
</ItemActions>
```

### ItemHeader

アイテムコンテンツの上にヘッダーを表示します。

```tsx
<Item>
  <ItemHeader>Header</ItemHeader>
  <ItemContent>...</ItemContent>
</Item>
```

### ItemFooter

アイテムコンテンツの下にフッターを表示します。

```tsx
<Item>
  <ItemContent>...</ItemContent>
  <ItemFooter>Footer</ItemFooter>
</Item>
```

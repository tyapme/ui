## 使い方

```tsx

  ColorField,
  ColorFieldGroup,
  ColorFieldInput,
  ColorFieldPrefix,
  ColorFieldSuffix,
} from "@/components/ui/color-field"
```

```tsx
<ColorField defaultValue="#3b82f6">
  <Label>カラー</Label>
  <ColorFieldGroup>
    <ColorFieldInput />
  </ColorFieldGroup>
</ColorField>
```

## API

### ColorField

| Prop           | Type                                                          | Default | Description              |
| -------------- | ------------------------------------------------------------- | ------- | ------------------------ |
| `value`        | `string \| Color \| null`                                    | -       | 制御された色の値         |
| `defaultValue` | `string \| Color \| null`                                    | -       | 初期値（非制御）         |
| `onChange`     | `(color: Color) => void`                                      | -       | 値変更時のコールバック   |
| `colorSpace`   | `"rgb" \| "hsl" \| "hsb" \| "hsl"`                           | -       | 入力するカラースペース   |
| `channel`      | `"red" \| "green" \| "blue" \| "hue" \| "saturation" \| ...` | -       | 編集するチャンネル       |
| `isDisabled`   | `boolean`                                                     | `false` | 無効状態                 |
| `isReadOnly`   | `boolean`                                                     | `false` | 読み取り専用             |
| `isInvalid`    | `boolean`                                                     | `false` | バリデーションエラー状態 |
| `className`    | `string`                                                      | -       | className                |

### ColorFieldGroup

`div` のすべての props を受け取ります。フォーカス・バリデーション状態のスタイルを管理します。

### ColorFieldInput

`react-aria-components` の `Input` props を受け取ります。

### ColorFieldPrefix / ColorFieldSuffix

`div` のすべての props を受け取ります。入力フィールドの前後に装飾要素を追加します。

## キーボード操作

| キー  | 動作               |
| ----- | ------------------ |
| `↑`   | 値を 1 増加        |
| `↓`   | 値を 1 減少        |
| `↑×10`| 値を 10 増加       |
| `↓×10`| 値を 10 減少       |

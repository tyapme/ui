## 使い方

```tsx showLineNumbers
import { Label } from "@/components/ui/label"
import { RadioGroup, RadioGroupItem } from "@/components/ui/radio-group"
```

```tsx showLineNumbers
<RadioGroup defaultValue="option-one">
  <div className="flex items-center gap-3">
    <RadioGroupItem value="option-one" id="option-one" />
    <Label htmlFor="option-one">Option One</Label>
  </div>
  <div className="flex items-center gap-3">
    <RadioGroupItem value="option-two" id="option-two" />
    <Label htmlFor="option-two">Option Two</Label>
  </div>
</RadioGroup>
```

## 構成

`RadioGroup` の構成:

```text
RadioGroup
├── RadioGroupItem
└── RadioGroupItem
```

## 使用例

### 説明

`Field` コンポーネントを使った説明付きのラジオグループ項目。

<ComponentPreview styleName="base" name="radio-group-description" />

### Choice Card

`FieldLabel` で `Field` 全体を包括してクリック可能なカード形式の選択屋を作成します。

<ComponentPreview styleName="base" name="radio-group-choice-card" />

### Fieldset

`FieldSet` と `FieldLegend` でラジオ項目をラベルと説明付きのグループにまとめます。

<ComponentPreview styleName="base" name="radio-group-fieldset" />

### 無効

`RadioGroup` の `disabled` で全項目を無効化します。

<ComponentPreview styleName="base" name="radio-group-disabled" />

### エラー

`RadioGroupItem` の `aria-invalid` と `Field` の `data-invalid` でバリデーションエラーを表示します。

<ComponentPreview styleName="base" name="radio-group-invalid" />

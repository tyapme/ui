## 使い方

```tsx showLineNumbers
import {
  Drawer,
  DrawerClose,
  DrawerContent,
  DrawerDescription,
  DrawerFooter,
  DrawerHeader,
  DrawerTitle,
  DrawerTrigger,
} from "@/components/ui/drawer"
```

```tsx showLineNumbers
<Drawer>
  <DrawerTrigger>Open</DrawerTrigger>
  <DrawerContent>
    <DrawerHeader>
      <DrawerTitle>Are you absolutely sure?</DrawerTitle>
      <DrawerDescription>This action cannot be undone.</DrawerDescription>
    </DrawerHeader>
    <DrawerFooter>
      <Button>Submit</Button>
      <DrawerClose>
        <Button variant="outline">Cancel</Button>
      </DrawerClose>
    </DrawerFooter>
  </DrawerContent>
</Drawer>
```

## 構成

`Drawer` の構成:

```text
Drawer
├── DrawerTrigger
└── DrawerContent
    ├── DrawerHeader
    │   ├── DrawerTitle
    │   └── DrawerDescription
    └── DrawerFooter
```

## 使用例

### スクロール可能なコンテンツ

コンテンツがスクロールしてもアクションを常に表示します。

<ComponentPreview styleName="base" name="drawer-scrollable-content" />

### 方向

`direction` prop でドロワーの表示方向を設定します。`top`、`right`、`bottom`、`left` が利用可能です。

<ComponentPreview styleName="base" name="drawer-sides" />

### レスポンシブダイアログ

`Dialog` と `Drawer` を組み合わせてレスポンシブなダイアログを作成できます。デスクトップでは `Dialog`、モバイルでは `Drawer` が表示されます。

## Cursor

Tailwind v4 ではボタンコンポーネントの `cursor: pointer` から `cursor: default` への[変更](https://tailwindcss.com/docs/upgrade-guide#buttons-use-the-default-cursor)が行われました。

`cursor: pointer` の振る舞いを維持したい場合は、CSS ファイルに以下のコードを追加してください:

`npx shadcn@latest init --pointer` でプロジェクト設定時にも有効化できます。

```css showLineNumbers title="globals.css"
@layer base {
  button:not(:disabled),
  [role="button"]:not(:disabled) {
    cursor: pointer;
  }
}
```

### As Link

`buttonVariants` ヘルパーを使ってリンクをボタンのように見せることができます。

**リンクには `<Button render={<a />} nativeButton={false} />` を使わないでください。** Base UI の `Button` コンポーネントは常に `role="button"` を適用するため、`<a>` 要素のセマンティックなリンクロールを上書きしてしまいます。代わりに `buttonVariants` と通常の `<a>` タグを使用してください。

## API リファレンス

### Button

`Button` コンポーネントは `button` 要素のラッパーで、各種スタイルと機能を追加します。

| Prop      | Type                                                                                 | Default     |
| --------- | ------------------------------------------------------------------------------------ | ----------- |
| `variant` | `"default" \| "outline" \| "ghost" \| "destructive" \| "secondary" \| "link"`        | `"default"` |
| `size`    | `"default" \| "xs" \| "sm" \| "lg" \| "icon" \| "icon-xs" \| "icon-sm" \| "icon-lg"` | `"default"` |

## 使い方

```tsx showLineNumbers
import { ScrollArea, ScrollBar } from "@/components/ui/scroll-area"
```

```tsx showLineNumbers
<ScrollArea className="h-[200px] w-[350px] rounded-md border p-4">
  Your scrollable content here.
</ScrollArea>
```

## 構成

`ScrollArea` の構成:

```text
ScrollArea
└── ScrollBar
```

## 使用例

### 横方向

`ScrollBar` に `orientation="horizontal"` を設定すると横方向スクロールが有効になります。

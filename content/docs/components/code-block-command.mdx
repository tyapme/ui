## 使い方

```tsx
import { CodeBlockCommand } from "@/components/ui/code-block-command"
```

```tsx
<CodeBlockCommand
  npm="npm install @base-ui/react"
  yarn="yarn add @base-ui/react"
  pnpm="pnpm add @base-ui/react"
  bun="bun add @base-ui/react"
/>
```

選択したパッケージマネージャーは `localStorage` に保存され、同一ページ内の複数インスタンスで同期されます。

## Props

| Prop | 型 | 説明 |
| --- | --- | --- |
| `npm` | `string` | npm 用コマンド |
| `yarn` | `string` | yarn 用コマンド |
| `pnpm` | `string` | pnpm 用コマンド |
| `bun` | `string` | bun 用コマンド |
| `className` | `string` | 追加の CSS クラス |

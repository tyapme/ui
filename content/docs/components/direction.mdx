`DirectionProvider` コンポーネントは、アプリのテキスト方向（`ltr` または `rtl`）を設定するために使用します。アラビア語・ヘブライ語・ペルシア語などの右対左言語をサポートする際に不可欠です。

RTL モードのプレビューです。言語セレクターで言語を切り替えられます。各コンポーネントページの RTL セクションでさらなる例を確認できます。

## 使い方

```tsx showLineNumbers
import { DirectionProvider } from "@/components/ui/direction"
```

```tsx showLineNumbers
<html dir="rtl">
  <body>
    <DirectionProvider direction="rtl">
      {/* Your app content */}
    </DirectionProvider>
  </body>
</html>
```

## useDirection

`useDirection` フックは、アプリの現在の方向を取得するために使用します。

```tsx showLineNumbers
import { useDirection } from "@/components/ui/direction"

function MyComponent() {
  const direction = useDirection()
  return <div>Current direction: {direction}</div>
}
```

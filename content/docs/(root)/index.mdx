TYAP.ME UI は、TYAP.ME の画面で使うコンポーネント、テーマ、フォーム、実装ルールをまとめたプロダクト UI の基盤です。

目的は、画面ごとの例外を増やさず、同じ役割の UI を同じ見た目と挙動で実装できる状態を保つことです。

## 方針

- **一貫性:** コンポーネント、余白、色、タイポグラフィを共通ルールに揃える。
- **再利用:** 同じ役割の UI はページ単位で作らず、共通コンポーネントとして扱う。
- **保守性:** 実装・テスト・ドキュメントを同じ変更単位で更新する。
- **プロダクト優先:** TYAP.ME の勤務管理体験に必要な UI だけを残す。

## 使い方

まず [Installation](/docs/installation) でプロジェクト設定を確認し、[Components](/docs/components) で利用できる UI を確認してください。

フォーム実装は [Forms](/docs/forms) にまとめています。
